# Einstein-Skill：アインシュタイン研究テイストモデリングシステム

> **史料に基づくアインシュタインの科学的研究テイストの計算モデリング**
> アインシュタインが候補となる科学理論をどのように評価したかを、歴史的証拠に基づいて評価・ランク付け・説明します。ロールプレイではありません。

## これは何ですか？

本システムはアインシュタインの**研究テイスト（Research Taste）**をモデル化します。つまり、異なる種類の科学理論に対する彼の文書化された好みです。

- *「1935年のアインシュタインは量子場理論をどう評価しただろうか？」*
- *「隠れ変数理論とコペンハーゲン解釈、アインシュタインはどちらを好んだか？」*
- *「数学的美への重視はキャリアを通じてどう変化したか？」*

**ロールプレイではありません。** すべての評価は、アインシュタインの論文、書簡、講演、および学術的分析からの歴史的証拠に基づいています。

## 主な特徴

- **8つのテイスト軸** — Holtonの主題分析、Howardの哲学研究に基づく
- **33の証拠レコード** — 一次資料（アインシュタインの著作）+ 二次学術分析
- **時間カットオフ** — 1900年から1955年の任意の時点で評価可能
- **証拠/推論の分離** — 各スコアは具体的な歴史的出典に追跡可能
- **RAGパイプライン** — 検索拡張生成により評価を実証拠に基づかせる

## 8つのテイスト軸

| # | テイスト軸 | 重み | 意味 |
|---|-----------|------|------|
| 1 | **不変性** | 0.95 | 物理法則はすべての座標系で同じ形式 |
| 2 | **統一性** | 0.90 | 異なる現象を単一の枠組みに統合 |
| 3 | **単純性** | 0.85 | 最小限の仮定で経験を十分に表現 |
| 4 | **物理的実在性** | 0.80 | 客観的実在は観測に依存しない |
| 5 | **因果的連続性** | 0.75 | 局所的で連続的な因果関係を好む |
| 6 | **数学的美** | 0.70 | 数学的優雅さが物理的真理を導く |
| 7 | **経験的基盤** | 0.65 | 理論は観測可能な現象と結びつく必要がある |
| 8 | **思考実験** | 0.60 | Gedankenexperimentを主要な方法論として |

## クイックスタート

```bash
git clone https://github.com/ezy1999/Einstein-Skill.git
cd Einstein-Skill
pip install -e ".[dev]"
einstein-taste fetch-data

# オプション：LLM評価用のAPIキー設定

# またはオフラインデモを実行

```
